--------------------------------------
           LISTAS
--------------------------------------

PARA CRIAR UMA LISTA USAMOS :


OL -  LISTA ORDENADA (ORDERED LIST)
------------------------------------

LI - ITEM LISTA (LIST ITEM)


EXISTEM PARAMETROS QUE PODE SER COLOCADOS 
EM OL PARA SER UMA LISTA ALFABÉTICA(A,B,C,
D..Z)  OU UMA LISTA NUMERIA (1,2,3,4...).

<OL TYPE='1>
<OL TYPE='A'>

PODEMOS DIZER TABEM PARA LISTA COMEÇAR EM 4 
BASTA USAR O PARAMETRO START

EX:

<OL TYPE='1' START='4'>



UL - LISTAS NÃO ORDENADAS (UNORDERED LIST)
--------------------------------------------

SEM NÚMERAÇÃO (ITENS QUE NÃO NECESSITAM SER
NUMERADOS)

TAMBEM PODES PASSAR O PARAMETRO DE TIPO PARA ELAS:

DISK - DISCO PRETINHO

CIRCLE - BOLINHA ABERTA

SQUARE - BOLINHA QUADRADINHA

<UL TYPE='CIRCLE'>

--RESPONSIVIDADE------------


PARA TORNAR UMA CAIXA FLEXIVEL USAMOS 2 COMANDOS:

MIN-WIDTH:;    - LARGURA MÍNIMA QUE UM ELEMENTO DEVE TER

MAX-WIDTH:;    - LARGURA MÁXIMA QUE UM ELEMENTO PODE CHEGAR


EX:

MIN-WIDTH:200PX;
MAX-WIDTH:1040PX;

ALGO IMPORTANTE DE CITAR É QUE, A CAIXA SE ADAPTA AO TAMNHO DA 
TELA ENTRE 200PX A 1040PX COMO FOI USADO NO EXEMPLO.




----PROPIEDADES BACKGROUND----

BACKGROUND-IMAGE:LINEAR-GRADINETE(TO BOTTOM COR 1, COR2,
 COR 3); - SREVE PARA FAZER UM DEGRADÊ COM 3 CORES OU MAIS
DE CIMA PARA BAIXO.

BACKGROUND-IMAGE : URL(LOCAL DO ARQUIVO) -  CARREGA UMA IMAGEM DE
                                           FUNDO.

BACKGROUND-IMAGE:REPEAT; - SERVE PARA FAZER UMA IMAGEM SE REPETIR
                         VÁRIAS VEZES NO FUNDO DO TEU SITE.

BACKGROUND-IMAGE:NO-REPEAT; - SERVE PARA UMA IMAGEM PARAR DE SE REPETIR
                            NO FUNDO DO SEU SITE.

BACKGROUND-SIZE: PX PX; - SERVE PARA DEFINIR A ALTURA E LARGURA DA SUA 
                        IMAGEM DE FUNDO.

-------- TARBALHANDO COM TABELAS--------


PARA CIRAR UMA TABELA USAMOS A TAG <TABLE>


A ESTRUTURA DE UMA TABELA BÁSICA É :  TABLE  - TABELA
                                          TR - LINHA
                                            TD - DADO


VERTICAL-AILING - DECIDE ONDE SEU TD - DADOS  IRAM FICAR 
SEJA NO TOPO OU EMBAIXO ETC.. DE FORMA VERTCAL


TABELAS GRANDES :

          THEAD - CABEÇALHO DA TABELA 
          TBODY - CORPO DA TABELA
          TFOOT - RODAPÉ DA TABELA

ABREVIAÇÕES

TH - ALGO IMPORTANTE

TD - UM DADO 

TR -  UMA LINHA


SCOPE DIZ A QUAL GRUPO SEU TITULO PERTENCE ATRAVÉS DE
ESPANSÃO.


COLSPAN - ESPANSÃO DE COLUNA

ROWSPAN - EXPANSÃO DE LINHA


COLGROUP


COL - PARA CADA COLUNA QUE VOCÊ TIVER EM SUA TABELA

      Get e Post
-------------------------

method='metodo'

Get - deixa os dados na URL (max-3000mil bytes)

post - deixa os dados "ocultos"

         TYPES
-----------------------------
reset - apagar /resetar dados

submit - enviar


parametros
-----------------------------

placeholder - dá uma dica do que é para fazer

size - tamanho 

required - requer algo

minlength='VALOR' - comprimento minimo

maxlength='VALOR' - comprimento máximo


CONTROLES DE FORMULÁRIO
----------------------------

MIN='VALOR' - MINIMO PARA CAIXA NUMÉRICA

MAX='VALOR' - MÁXIMO PARA CAIXA NUMÉRICA

STEP='VALOR' - DE QUANTO EM QUAnTO SERÁ 
              DIGITADO
